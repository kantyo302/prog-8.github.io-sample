HTML & CSS

WEBページを作成するのに、使用される言語です。HTMLとCSSを組み合わせることで、静的なページを作り上げることができます。

JQUERY

素敵な動きを手軽に実装できるJavaScriptライブラリです。アニメーション効果をつけることができ、動的なページを作ることができます。

Ruby

オープンソースの動的なプログラミング言語で、シンプルさと高い生産性を兼ね備えています。大きなアプリケーションから小さな日用ツール、さまざまなソフトウェアを作り上げることができます。

Java

HTMLだけではページの内容を変更できません。PHPはHTMLにプログラムを埋め込み、それを可能にします。
